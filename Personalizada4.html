Las motos de alto cilindraje cuentan con el sistema de Fuel Injection o inyección electrónica. En este sistema se obtiene una alta precisión al momento de administrar el combustible al motor, esto no aplica para las carburadas, ya que no se tiene un control del suministro, lo que muchas veces genera más consumo.
